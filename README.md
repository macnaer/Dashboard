# Dashboard application with using React, TypeScript, Redux, Express and MS SQL

## Install node_modules for Front-End and Back-End

In the project directory, run:

### `npm i && npm i --prefix client-app`

## Available Scripts

Development mode:\

### `npm run dev`

Run server:\

### `npm run start`

Run React app:\

### `npm run client`

Run Seeder.js:\

### `npm run seed`
